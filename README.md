#  Synent Task 3 – Customer Segmentation

##  Problem Statement

The objective of this project is to segment customers into distinct groups based on their characteristics and purchasing behavior. Customer segmentation helps businesses understand different customer profiles and make data-driven marketing and business decisions.

---

##  Dataset Details

**Dataset:** Customer Personality Analysis Dataset (Kaggle)

**Description:**
The dataset contains customer demographic information, income, spending habits, and campaign response data.

* **Rows:** ~2,200+
* **Columns:** 29

---

##  Approach

The following steps were performed:

* Loaded and explored the dataset.
* Cleaned and preprocessed the data.
* Selected relevant features for segmentation.
* Scaled the data using feature normalization.
* Applied the **K-Means Clustering** algorithm.
* Visualized customer clusters using scatter plots and evaluated the segmentation.

---

##  Results

*  Successfully grouped customers into meaningful clusters.
*  Identified customer segments based on purchasing behavior.
*  Visualized clusters for easy interpretation.
*  Generated insights to support targeted marketing strategies.
